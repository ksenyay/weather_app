# 🌤️ Weather Dashboard

A sleek and modern weather dashboard built with **vanilla JavaScript**, offering real-time weather updates and customizable themes. View daily and hourly forecasts, detailed weather metrics, and enjoy a beautiful interface with scenic backgrounds.

![image_2025-05-23_14-45-05](https://github.com/user-attachments/assets/93db6482-8181-4437-839b-4ccb843d45a5)

## 🚀 Features

- 📍 **Real-Time Weather Data**  
  Fetches current, hourly, and daily weather data from a live API.

- 🎨 **Customizable Themes**  
  Switch between different background themes like "Mountains", etc. for a more personalized experience.

- 🌡️ **Detailed Forecast**  
  - Current temperature, weather conditions, and "feels like"
  - Hourly forecasts
  - Wind, precipitation, humidity, UV index, and rain chance

## 🔧 Technologies Used

- **HTML5**  
- **CSS3**   
- **JavaScript (ES6+)**  
- **Weather API ([weatherapi.com](https://www.weatherapi.com/))**
